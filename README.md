# 3D-visualisointi

## Projektin kääntäminen ja suorittaminen

Tarvittavat työkalut:

- Java SDK 21
- [sbt](https://www.scala-sbt.org/)

projektin voi kääntyy ja käynnistää komennolla `sbt run`.
